## 概述

### 定义

四个能力等级定名为 0 级至3 级，每一级是一个层次，作为继续进行过程改进的基础

0. 不完整级
1. 执行级
2. 已管理级
3. 已定义级

成熟度级别

1. 初始级
2. 已管理级
3. 已定义级
4. 已量化管理级
5. 持续优化级。
