## 概述

面向对象**设计**的基本任务是把`面向对象分析模型`转换为`面向对象设计模型`

面向对象的分析模型：主要由**顶层架构图**、**用例与用例图**、**领域概念模型**构成

设计模型则包含以`包图`表示的**软件体系结构图**、**以交互图表示的用例实现图**、**完整精确的类图**、**针对复杂对象的状态图**和**描述流程化处理过程的活动图**等
